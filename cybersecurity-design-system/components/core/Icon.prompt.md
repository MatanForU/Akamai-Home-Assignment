Thin-stroke line icon from Lucide (CDN-hosted, MIT license) — the product's sole icon system. Covers security/ops glyphs well: `shield`, `shield-check`, `shield-alert`, `server`, `lock`, `key`, `bug`, `activity`, `radar`, `alert-triangle`, `search`, `filter`, `bell`, `settings`, `layout-dashboard`.

```jsx
<Icon name="shield-check" size={16} color="var(--accent-primary)" />
```

Rendered via a CSS `mask-image`, so `color` fully re-tints the glyph — safe to use `currentColor` (default) or any token, in both light and dark mode.
